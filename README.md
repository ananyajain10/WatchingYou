# Watching You

A productivity and daily tracking mobile app built with React Native. **Watching You** helps users organize their day with tasks, notes, focused work sessions, and end-of-day summaries.

---

## Features

### Calendar Screen
- Navigate through dates easily
- View and manage tasks for any selected day
- Clean and simple daily planning experience

### Task Cards
- Add task cards for each day
- Organize goals, priorities, and to-dos
- Update progress anytime

### Notes for Every Task
- Add notes to each individual task
- Save reminders, ideas, or task-specific details
- Keep all context attached to the task

### Timer (Start / Stop / Continue)
- Start a timer for any current-day card
- Stop the timer whenever needed
- Continue paused sessions later
- Great for focus sessions and productivity tracking

### Day Summary
- Generate an end-of-day summary
- Review completed work and progress
- Show summary in a shareable popup
- Share your day with others

---

## Tech Stack
- React Native
- JavaScript / TypeScript
- React Navigation
- AsyncStorage / Local Storage
- Custom Components & Hooks

---

## Installation

```bash
# Clone the repository
git clone <your-repository-url>

# Move into project folder
cd watching-you

# Install dependencies
npm install

# Run on Android
npx react-native run-android

# Run on iOS
npx react-native run-ios
```
